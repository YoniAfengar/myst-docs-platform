# 🚀 Getting Started

This guide explains how to start the project locally.

## Prerequisites

- Docker
- Docker Compose
- Git

## Clone

```bash
git clone https://github.com/YoniAfengar/myst-docs-platform.git

cd myst-docs-platform
```

## Run

```bash
make up
```

Open:

```text
http://localhost:3001
```
