# 🚀 Getting Started

## Prerequisites

Before you begin, install:

- Docker
- Docker Compose
- Git

## Clone the repository

```bash
git clone https://github.com/YoniAfengar/myst-docs-platform.git

cd myst-docs-platform
```

## Start the documentation

```bash
make up
```

Open:

```text
http://localhost:3001
```
