# 🚀 MyST Docs Platform

> **A production-ready documentation platform built with MyST and Docker.**

Welcome to **MyST Docs Platform** — a modern documentation project designed to demonstrate how to build, organize, and maintain professional technical documentation using **MyST**, **Docker**, and modern development practices.

---

## ✨ Why this project?

This repository is more than a simple MyST example.

It demonstrates how to build a reusable documentation platform that can serve as a foundation for real-world software projects.

Whether you're documenting an application, a data pipeline, or an internal engineering system, this project provides a clean starting point.

---

## ⚡ Quick Start

Clone the repository:

```bash
git clone https://github.com/YoniAfengar/myst-docs-platform.git
cd myst-docs-platform
```

Start the development environment:

```bash
make up
```

Open your browser:

```text
http://localhost:3001
```

That's it.

---

## ✨ Features

- 🐳 Docker-based development environment
- ⚡ Live reload while editing documentation
- 📚 Multi-page documentation structure
- 📝 MyST Markdown
- 📊 Mermaid diagrams
- 📓 Jupyter Notebook support
- 📄 HTML & PDF export
- 🛠 Developer-friendly workflow
