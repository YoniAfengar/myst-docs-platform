# 🚀 MyST Docs Platform

> A production-ready documentation platform built with **MyST**, **Docker**, and modern documentation best practices.

---

## 📖 Overview

**MyST Docs Platform** is a hands-on project that demonstrates how to build a modern technical documentation platform using MyST.

The project focuses on:

- 📚 Structured technical documentation
- 🐳 Docker-based development
- ⚡ Live preview and rapid iteration
- 🧩 Modern documentation workflows
- 🧪 Interactive MyST feature demonstrations

Rather than being just a collection of Markdown files, this project aims to serve as a reusable foundation for future documentation projects.

---

## ✨ Features

- 🐳 Docker development environment
- ⚡ Live preview
- 📝 MyST Markdown
- 📚 Multi-page documentation
- 📊 Mermaid diagrams
- 🧮 Mathematical equations
- 💻 Syntax highlighting
- 🧪 MyST Playground
- ⚙️ Makefile automation

## 🚀 Quick Start

Clone the repository:

```bash
git clone https://github.com/YoniAfengar/myst-docs-platform.git
cd myst-docs-platform
```

Start the project:

```bash
make up
```

Open your browser:

```text
http://localhost:3001
```
